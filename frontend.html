FRONTEND


Se refiere a la práctica de producir HTML, CSS y JavaScript. Estos tres elementos se encargan de dar forma a la parte frontal de un sitio web o aplicación. Esto incluye los fondos, colores, texto, animaciones o efectos.


FUNDAMENTOS:


  • HTML: Es el lenguaje de marcado que estructura el contenido en la web. Define los elementos de la página como encabezados, párrafos, listas, enlaces, y otros.

  • CSS: Es el lenguaje utilizado para definir la presentación visual de los documentos HTML. Permite controlar colores, fuentes, márgenes, disposición y diseño.

  • JavaScript: Es el lenguaje de programación que permite agregar interactividad y dinamismo a las páginas web. Se usa para manipular el DOM (Document Object Model), manejar eventos, realizar validaciones, y más.
